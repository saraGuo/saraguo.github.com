慧慧：

以后这个网站就是专属你了。可以通过: http://isara.cn 或者 http://guohuizhong.com 都可以访问，你可以将你的网站打扮成任何你喜欢的样子（我教你详细办法）。可以在上面发照片、旅游心得、产品心得、开心的、难过的。

之后当你想写文章时，你只需要3步即可：

  • 在文件新建一个文件，例如new-article.md
  • 打开文件，在里面按照markdown格式写文章了。
  • 写完之后执行一条命令就可以轻松发送到网上了hexo d -g
    是不是很简单^-^

The first using

  1. 点击我下载NODEJS
  2. 点击我下载MYGIT
  3. 随便进入一个文件夹，按住shift键，同时按下鼠标右键，选择此处打开命令行
  4. 在黑色窗口输入：npm install -g hexo，并按下回车键
  5. 输入git clone https://github.com/saraGuo/sara.git，并按下回车键
  6. 输入cd sara，并按下回车键
    …
  7. 写不下去了，还是我帮你安装吧，操作一次之后才能写完
